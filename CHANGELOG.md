# Change Log

Toutes les modifications notables de l'extension "zui-helper" seront documentées dans ce fichier.

## [0.0.1] - Initial Release

- **IntelliSense** : Ajout de l'auto-complétion pour toutes les méthodes `zUI` en Lua.
- **Hover Documentation** : Documentation complète et exemples au survol des fonctions.
- **Signature Help** : Aide contextuelle pour les paramètres des fonctions lors de la frappe.
- **Snippets** : Collection complète de snippets (`zmenu`, `zbtn`, `zlist`, etc.) pour une création rapide.
- **Commandes** : Ajout de la commande `zUI: Insert Menu Template` pour insérer des modèles de menus pré-configurés.
- **Color Picker** : Visualisation et édition des codes couleurs hexadécimaux directement dans l'éditeur Lua.
